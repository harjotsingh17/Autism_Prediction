# 🧠 Autism Prediction using Machine Learning

A Machine Learning web application built with **Streamlit** that predicts the likelihood of Autism Spectrum Disorder (ASD) based on user-provided information.

The project includes data preprocessing, model training using multiple machine learning techniques, and an interactive web interface for making predictions.

---

## 🚀 Features

- Interactive Streamlit web application
- Machine Learning-based autism prediction
- Data preprocessing pipeline
- Model training script
- Easy-to-use prediction interface
- Modular project structure
- Clean and responsive UI

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Autism-Prediction.git
```

Move into the project folder

```bash
cd Autism-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

### Step 1: Train the Model

```bash
python train_model.py
```

### Step 2: Launch the Streamlit Application

```bash
streamlit run app.py
```
